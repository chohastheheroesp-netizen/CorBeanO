Developer Program Requirements

Our 6-month junior developer programme is designed for ambitious newcomers to tech. Review the requirements below and use our checker to see which learning path suits you best.

General Requirements

  • ✓ Minimum age: 18 years old
  • ✓ No coding experience necessary — we teach from scratch
  • ✓ Full-time commitment (37.5 hours/week) for 6 months minimum
  • ✓ Genuine passion for coding and willingness to learn continuously

Learning Path Prerequisites

  • Web Fundamentals: None — starts week 1 for all developers
  • JavaScript Mastery: Must complete Web Fundamentals (weeks 3–4)
  • React & Modern Frontend: Must complete JavaScript Mastery (weeks 5–8)
  • Backend Development: Must complete React AND JavaScript (weeks 9+)
